# WS-Skill09-Sao-Paulo-Project-Session-7

# Requirements
To start the application a local installation of mssql db is neccessary. If this is not installed on your device please add it. To add the database use the Database1.mdf file located in the database folder and attach it to your local db under the name 'MarathonSkillsDB'. It is important that the name is correct (including upper and lower case). After this is done the application can be started. To import staff data use the green button on the main page to do so.

# Starting the application
Navigate into the Executable/net9.0-windows10.0.19041.0/win10-x64 folder and double click the 'Session-7-Dennis-Hilfinger.exe' file to start the application.
